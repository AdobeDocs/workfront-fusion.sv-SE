# Versionsaktivitet för Workfront Fusion: 16 februari 2026

Den här sidan beskriver alla förbättringar som gjorts i Adobe Workfront Fusion den 16 februari 2026.

En lista med alla senaste ändringar finns i [Adobe Workfront Fusion-versionsaktivitet](/help/workfront-fusion/fusion-product-releases/fusion-release-activity.md).

En lista med de senaste felkorrigeringarna i Workfront Fusion finns på sidan [Workfront Maintenance Updates](https://experienceleague.adobe.com/sv/docs/workfront-known-issues/releases/current-updates) och sök efter uppdateringar med namnet Workfront Fusion Maintenance Update.

## Lägg till AI-uppmaningar i dina scenarier med den nya modulen MCP Agent

Du kan inkludera en AI-prompt i ditt scenario med hjälp av Model Context Protocol (MCP) kombinerat med stora språkmodeller (LLM). Genom att konfigurera dessa i modulen MCP Agent kan du använda artificiell intelligens för att skapa arbetsflöden som är effektiva, säkra och flexibla.

Med MCP kan du använda en AI-prompt för att ansluta till program. Eftersom frågan kan ta hänsyn till variationer och sammanhang minskar detta behovet av att manuellt konfigurera vissa arbetsflöden, vägar eller scenarioavsnitt i Fusion.

Mer information om hur du använder AI-uppmaningar i Fusion finns i [Lägga till en AI-fråga i ditt scenario](/help/workfront-fusion/create-scenarios/add-modules/add-an-ai-prompt-to-your-scenario.md).

Mer information om hur du konfigurerar modulen MCP Agent finns i [modulen MCP Agent](/help/workfront-fusion/references/apps-and-modules/tools-and-transformers/model-context-protocol-mcp-connector.md).

## Visa prestanda för hela organisationen på Fusion Performance Dashboard

För att göra det enklare att förstå Fusion-prestanda i hela organisationen har vi skapat den nya Performance Dashboard-panelen. Nu kan Fusion-administratörer snabbt se vilka scenarier som körs mest, var förseningar inträffar och hur effektivt arbetarpooler fungerar. Med Performance Dashboard kan du se körningsvolymer, ködjup, poolanvändning och prestanda på scenarienivå i realtid.

Mer information om kontrollpanelen för Fusion-prestanda finns i [Visa kontrollpanelen för prestanda för en organisation](/help/workfront-fusion/set-up-and-manage-workfront-fusion/set-up-and-manage-orgs-and-teams/set-up-orgs-teams-and-users/view-performance-dashboard.md).
